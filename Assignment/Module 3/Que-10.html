Que-10 How can we club two or more rows or columns into a single row or column in an HTML table? With Example.

Using colspan and rowspan we can club two or more rows or columns into a single row or column in a HTML table. For Example:-

RowSpan

No Name Department Gender Salary
1 Jayden CSE Male 40000
2 Eliana IT Female 350000
3 Rowan Male


ColSpan

No Name Department Gender Salary
1 Jayden CSE Male 40000
2 Eliana IT Female 350000
3 Rowan IT Male 40000
Total-3
