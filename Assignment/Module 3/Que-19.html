Que-19 What is difference between HTML and XHTML?

HTML

Stand for HyperText Markup Language.
HTML is not case sensetive.
In HTML not need to close tag like: br,hr,meta,.
In XHTML it is not necessary to follow structure like html,head,title,body,etc.
HTML is more use than XHTML
It is not strict.



XHTML

Stand for Extensible HyperText Markup Language.
XHTML is a case sensetive.
In XHTML it is necessary to close the tag like br,hr,etc. In XHTML it is necessary to follow structure like html,head,title,body,etc.
It is stricter than HTML.
