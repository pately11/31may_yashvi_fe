Que-17 What is the meaning of the CSS selector?

CSS selector select the specific part what you want to style. Select by id, class, element, universal selector and many more.

EXAMPLE:-
<style>
.abc{
color: blue;
font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
}
</style>
