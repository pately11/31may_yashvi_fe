Que-19 What is the rule set?

In CSS a rule set is a combination of a selector and a declaration block.

Selector:

The selector indicates which HTML elements the rule set should apply to. For example, h1 selects all h1 elements, .class selects all elements with a specific class, #id selects an element with a specific ID, and so on.

Declaration Block:

The declaration block is enclosed within curly brackets {} and contains one or more property-value pairs separated by semicolons. For example, color: red; sets the text color to red, and font-size: 16px; sets the font size to 16 pixels.

Example:-

<style>
{
color: blue;
font-size: 24px;
font-weight: bold;
}
{
background-color: #4CAF50;
color: white;
padding: 10px 20px;
text-decoration: none;
display: inline-block;
font-size: 16px;
border-radius: 5px;
}

</style>
