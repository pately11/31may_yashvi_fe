Que-3 What is the difference between CSS2 and CSS3?

CSS2 CSS3
Basic selectors like element selectors, class selectors and id selectors. Advanced selectors like attribute selectors,
pseudo class selector (Focus, Hover, Link and nth child), pseudo element selectors (After, Before, First-line,
First-letter, Selection).
In css2 cocept of box model, model's properties are height, width, margin and padding. CSS3 give advanced properties in box model like box-sizing, border-radius, box-shadow.
Basic layout like float and positions. Advanced layout like flex and grid.
Didn't support Animation and Transition. Support Animation and Transition.
Didn't have media query support. Support media query to make design responsive.
Allow only single background. Allow multiple backgrounds.
